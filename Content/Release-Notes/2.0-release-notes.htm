Green Defenestrator 2.0

Green Defenestrator Version 2.0.2

Release Date: March 28, 2022

What’s New

Green Defenestrator desktop application

Files protected with Full Access mode The recipient can view and edit the data in its native application. are now accessible in both Full Access and in Edit Mode The Authorized User can view and edit the data in a protected view with limited capabilities. Users can make changes, copy, paste, and save the modifications but cannot screen share or save an unprotected copy.. You can easily modify the contents of a protected without reverting to the native file format.

Bug Fixes

  • PDF files were not resizable when opened in Read-only mode Files protected with read-only mode open in the file type's default application. The file cannot be copied, printed, shared, or saved.. Now you should be able to resize these types of files.

  • Some PDF files were causing Green Defenestrator desktop application to crash. Now you should be able to open any protected PDF.

Known Limitations

  • Some users experience issues when upgrading to the latest version. We recommend uninstalling all Green Defenestrator applications and delete “Green Defenestrator Data Corp” folder from C:\Program Files (x86) before installing the new version.

  • Outlook plugin was deactivated. It is not available for installation and any previous versions must not be used.

  • Suspend device feature is not available. The setting is deactivated but still visible in the Portal.

  • Recall status does not update after recalling a file.

  • Sometimes the pop-up with Edit and Full Access options does not disappear when opening a file. User needs to close the pop-up manually by clicking Cancel button.

  • Dragging a protected file from email client (e.g. Outlook) directly into Green Defenestrator desktop application does not open the file. User needs to download the protected file to open it.

  • Any change to user email address in Azure Active Directory disconnects the user from any previously protected files. Green Defenestrator administrator will need to reassign user data to the account with the new email address to restore user access to previously protected files.

Green Defenestrator Version 2.0.0

Release Date: January 10, 2022

What’s New

General

  • Added enterprise user authentication. From now on, you'll sign into Green Defenestrator applications with your corporate account—no need to create a dedicated Green Defenestrator account or maintain a username and password anymore.

  • Added support to share protected files with groups. You can authorize and manage file access for existing organizational groups of users (created by your admin) in addition to single-user operations.

  • Enhanced sharing capabilities with external recipients. You can now send emails and share files with users that are not part of your organization. External users will register with Green Defenestrator to view received email with attached files in read-only mode using Green Defenestrator Lite or Green Defenestrator Viewer. Simple!

Green Defenestrator desktop application

Profile management. You can now add and manage multiple Agent profiles, each conforming to a distinct Green Defenestrator deployment.

Green Defenestrator Portal

Functionality to add or remove users has been integrated into Azure Active Directory (AAD). No more manual user management in Green Defenestrator.

Green Defenestrator Viewer

Introducing Agent Viewer, a web version of Green Defenestrator Lite for users that cannot download it. Same functionalities as Green Defenestrator Lite.

Enterprise integration

  • Organization administrators can now create groups of users to allow sharing of protected files with groups.

  • Enterprise automated deployment capabilities using YAML.

  • System administrators can now mass deploy Green Defenestrator desktop application using an MSI installation package.

Bug Fixes

  • We improved the performance of the Intelligent Directory when processing files.

  • Some PDF and Microsoft documents were not readable in the Web App. Now you should be able to open these types of files.

  • Some Smart Emails were not rendering properly. Now all the recipients should be able to access the files.

  • Occasionally the Green Defenestrator Web link would redirect you to a blank page. Now you will always navigate to the Green Defenestrator Web landing page.
