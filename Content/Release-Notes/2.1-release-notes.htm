Green Defenestrator 2.1

Green Defenestrator 2.1.0

Release Date: August 2022

What’s New

  • Read-only mode Files protected with read-only mode open in the file type's default application. The file cannot be copied, printed, shared, or saved. is now available for over 150 file types. You can protect all these file types across all Green Defenestrator products. All these file types can be opened in Green Defenestrator Desktop app. The full list of supported file types is available here.

  • Green Defenestrator now provides comprehensive documentation of logs (user actions and user-triggered events). This lets you create and export audit and forensics reports.

  • External (unlicensed) users can now open standalone protected files using Green Defenestrator Viewer. Previously, it was possible to open files only from a protected email.

  • All Microsoft plugins are deactivated and unsupported. We now provide a seamless experience with the support of native applications.

Bug Fixes

  • Some Green Defenestrator security groups didn’t have the correct scope of rights. That issue has been fixed in the following ways:

  • Read-only users can’t protect files.

  • Only non-admin users can protect and open files.

  • Security groups now have cumulative capabilities.

  • Green Defenestrator Mobile apps were removed from the Apple Store and Play Store. In the future, Mobile applications will be delivered directly to customers.

  • Some users experienced issues when retrieving large number of files in Green Defenestrator Portal. Now, you may retrieve any number of files.

Known Limitations

  • In Read-only mode, Green Defenestrator can only open 3-D files up to 250MB in size and thumbnails or image previews if the image size is less than 100 MB (approximately 12000 x 8000 pixels). The full list of impacted file extensions is available here.

  • The user sees an error message when opening a file not supported in Read-only mode (you can still grant Read-only access to any file). The full list of supported file types is available here.

  • When opening certain type file in Read-only mode, clicking on some buttons (e.g. Share) will lead to an unexpected behavior (e.g. a share link pop-up not displaying properly). Usually, it’s possible to go back to the initial screen by pressing the Esc button.

  • Access time violations aren't shown in the Portal/Dashboard. This will be fixed in Green Defenestrator 3.0.

Green Defenestrator 2.1.0 macOS

The Green Defenestrator Desktop Application is now available for macOS users. You can now protect and open Green Defenestrator-enabled files on macOS devices, as well as manage Green Defenestrator User Profiles. Here are the features:

  • Read-only mode supports Microsoft Office files (except for PowerPoint), text, images, and Adobe PDF. A full list of supported file types is available here.

  • Full Access mode The recipient can view and edit the data in its native application. supports all file extensions

  • Edit mode The Authorized User can view and edit the data in a protected view with limited capabilities. Users can make changes, copy, paste, and save the modifications but cannot screen share or save an unprotected copy. is not supported for macOS. It will be available in future releases

Known Limitations

  • The app login page has a browser check. If the browser isn’t Safari, Edge, Chrome, or a Chromium-based browser, an “unsupported browser” message is displayed.

  • Offline mode is not supported (the icon at bottom is invisible) in Green Defenestrator Desktop app for macOS. You'll need to use Green Defenestrator Desktop for Windows to open files with offline access.

  • Files with Edit access open in Read-only mode in Green Defenestrator Desktop for macOS. You’ll need to use Green Defenestrator Desktop for Windows to open these files in Edit mode.
