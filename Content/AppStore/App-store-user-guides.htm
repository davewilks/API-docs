Keyavi Windows App Store Submission Testing Guide

The Keyavi Windows app enables you to protect files on your Windows device, send protected files to recipients, open protected files others have given you access to, and manage your protected files.

Set Up the Keyavi App

Before you start, find the organization code provided by the Keyavi administrator. If you don't have an organization code yet, contact the Keyavi administrator. To set up the Keyavi app Lightweight software interface that provides a user-friendly, intuitive interaction with Keyavi's technology. It allows users to create new and open intelligent files and manage settings easily., you'll add a default profile using the organization code and register your computer.

To set up the Keyavi app:

  1. Open Keyavi.

    Keyavi launches the accounts screen. If you don't see it, follow these steps to open the Accounts screen:

  2. Click the profile picture, select Profiles, and then select Manage.

  3. Click Add.

  4. Enter a descriptive and meaningful profile name, for example, Default or Keyavi USA. You can create multiple profiles, so keep the naming meaningful but straightforward.

  5. Paste the organization code that you received from your Keyavi administrator.

  6. Click Save. The profile is automatically set as your default profile. When you have multiple profiles, you'll be able to specify which profile you want as the default.

  7. Sign in using the information provided by your Keyavi administrator.

    Depending on how the organization configured Keyavi, you'll sign in with your organization's credentials or have the option to choose which credentials you use. You can use only one account per service.

  8. Click Yes when prompted to register your computer.

Keyavi is ready to go. You now see the main menu of the app with the Protect, Open, Manage, and Settings options.

Protect a File

You may protect a file stored locally on your device or in the cloud by using the following process.

If you're logging in with Microsoft, Google, or Apple credentials, you won't have access to the Protect or Manage menus.

  1. Open the Keyavi app and click Protect. On the Protect window in Keyavi, click Select File, and then select the file you want to protect.

    After selecting a file to protect, Keyavi shows the file on the Protect screen where you can add authorized users or groups (Add Authorization), select more files, or protect the file so only you have access.

  2. Tap or open the menu , and then tap Protect.

  3. On the Protect page in Keyavi, click Select File, and then select the file you want to protect.

    After selecting a document or folder to protect, Keyavi shows the files on the Protect screen where you can add authorized users or groups (Add Authorization), add a time embargo, select more files, or protect the file so only you have access.

  4. On the Protect page in Keyavi, tap Protect a new item, and then select the file you want to protect.

  5. Use the menu at the top to locate file locations on your device.

  6. Select the files to protect, and then tap Open.

    After selecting a document or folder to protect, Keyavi shows the files on the Protect screen where you can add authorized users or groups (Add Authorization), add a time embargo, select more files, or protect the file so only you have access.

  7. Add authorized users or groups. On the Protect window, click Add Authorization.

  8. Click Add to open the Access Authorization window where you can add users and groups.

  9. Use the Groups, Users, and External categories to navigate to the different user types where you can search or browse.

  10. After you find a user or group, select the access level (e.g., Read-only, Edit, or Full Access), and then click Done.

  11. After you find a user or group, tap Select, select the access level (e.g., Read-only, Edit, or Full Access), tap OK, and then click Done.

  12. Add a restrictive time period—also called time embargo—for when access to data begins and expires. For example, you can limit access to before and after a business meeting. Click Select Policy.

    1. In Access BeginsAccess Start, select the calendar icon to set the earliest date and time the data becomes accessible to authorized users. Then click CloseOK. The start date must be before the end date.

    2. In Access Ends, select the calendar icon to set the expiration date and time the data becomes inaccessible to authorized users. Then click CloseOK. The end date should be after the start date.

  13. Click Protect.

  14. Send or save the Keyavi-protected file using your device's sharing features. Unauthorized users can't access the file and will get a message with the option to request access.

At the top of the app, Keyavi shows a status message. A new copy of your data is created in the same folder as the original file with the IKD file Short for "Intelligent Keyavi Data" and is the file extension for a Keyavi-protected file. This file has multiple layers of encryption and policy-based rulesets infused into the data itself, making it self-protecting, intelligent, and self-aware. extension that you can share with authorized users. Unauthorized users, however, can't access the file and will get a message with the option to request access.

Open a Keyavi-protected File

If you receive a file with the .ikd file extension, you can download it on your device and use the Keyavi app to open (decrypt) it. On Windows and Mac, you can open the file in Keyavi by double-clicking it or right-clicking it and selecting Keyavi.

  1. Click Open.

  2. Tap or open the menu , tap Open, and then tap the plus button .

  3. Find and select the file, and then click Open. Note that you will only see IKD files.

    Find and tap the file. Note that only IKD files are available to select.

    • If you were authorized with Read-only or Edit access, the file opens in a Keyavi viewer. You won't be able to copy, print, share the content, or save an unprotected copy of the file.

    • If you were authorized with Full Access, choose how you want to open the file. You won't be able to change the protected copy of this file. You'll need to save an unprotected copy of the file to make changes.

Manage Your Protected Files

Click Manage to view your Keyavi Portal dashboard and Data Management pages. You can view your list of protected files and their metadata, including their chain of custody.

If you log in with Google, Microsoft, or Apple accounts, Manage is unavailable.
