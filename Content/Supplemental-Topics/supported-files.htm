Supported Files

Keyavi supports most common file types. Check this page for details about which fie types are supported in Read-only, Edit, and Full Access modes on your device.

Read-Only Mode

Keyavi supports these file types in Read-only mode Files protected with read-only mode open in the file type's default application. The file cannot be copied, printed, shared, or saved., which open in a Keyavi editor. The files cannot be copied, shared, or edited.

The Keyavi Windows app supports over 150 file types if configured for Native file type support.

Full Access Mode

Keyavi supports these file types in Edit mode The Authorized User can view and edit the data in a protected view with limited capabilities. Users can make changes, copy, paste, and save the modifications but cannot screen share or save an unprotected copy., which open the file's default editor.
