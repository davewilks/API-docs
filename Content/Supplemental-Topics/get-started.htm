Get Started

Companies use Keyavi to protect shared files within their organization using company credentials. They also share files outside of the company using credentials from services like Google, Windows, and Apple.

Whether you're an internal user with the company using your organization's credentials, or an external user signing in with a service account like Google, you'll use the Keyavi app Lightweight software interface that provides a user-friendly, intuitive interaction with Keyavi's technology. It allows users to create new and open intelligent files and manage settings easily. to access the Keyavi-protected files. Your Keyavi administrator provides an organization code that you'll use to set up your Keyavi app.

Get the Keyavi Apps

Depending on how your organization uses Keyavi, you might already have Keyavi on your computer or device. But if you don't have a Keyavi app, you can download it from these locations:

  • Keyavi Desktop for Windows from the Windows store.

  • Keyavi Desktop for Mac from the Apple App Store.

  • Keyavi iOS app from the Apple App Store.

  • Keyavi Android App from the Google Play Store.

Depending on how the organization configured Keyavi, you'll sign in with your organization's credentials or have the option to choose which credentials you use. See Sign in to Keyavi for more information.

Set Up the Keyavi App

Windows and Mac

Before you start, find the organization code provided by the Keyavi administrator. If you don't have an organization code yet, contact the Keyavi administrator. To set up the Keyavi app, you'll add a default profile using the organization code and register your computer.

To set up the Keyavi app:

  1. Open Keyavi.

    Keyavi launches the accounts screen. If you don't see it, follow these steps to open the Accounts screen:

  2. Click the profile picture, select Profiles, and then select Manage.

  3. Click Add.

  4. Enter a descriptive and meaningful profile name, for example, Default or Keyavi USA. You can create multiple profiles, so keep the naming meaningful but straightforward.

  5. Paste the organization code that you received from your Keyavi administrator.

  6. Click Save. The profile is automatically set as your default profile. When you have multiple profiles, you'll be able to specify which profile you want as the default.

  7. Sign in using the information provided by your Keyavi administrator.

    Depending on how the organization configured Keyavi, you'll sign in with your organization's credentials or have the option to choose which credentials you use. You can use only one account per service.

  8. Click Yes when prompted to register your computer.

Keyavi is ready to go. You now see the main menu of the app with the Protect, Open, Manage, and Settings options.

iOS and Android

Before you start, find the organization code provided by the Keyavi administrator. If you don't have an organization code yet, contact the Keyavi administrator. To set up the Keyavi app, you'll add a default profile using the organization code and register your computer.

To set up the Keyavi app:

  1. Open Keyavi.

    Keyavi prompts you to add a new profile.

  2. Tap Add.

  3. Enter a descriptive and meaningful profile name, for example, Default or Keyavi USA. You can create multiple profiles, so keep the naming meaningful but straightforward.

  4. Paste the organization code you received from your Keyavi administrator.

    If you are already set up on Windows or Mac you can scan the org code. Go to the Settings page on your computer, and then click Configure Mobile. On your mobile device, tap Scan Org Code and scan the code in Keyavi.

  5. Tap Add.

  6. Sign in using the information provided by your Keyavi administrator.

    Depending on how the organization configured Keyavi, you'll sign in with your organization's credentials or have the option to choose which credentials you use.

Keyavi is ready to go. Open the menu to see your options: Protect, Open, Manage, and Settings options.
