Manage Profiles

Profiles define your connection to an organization’s protected data. Each profile is associated with an organization code. You can share Keyavi-protected data only with other users who use the same organization code in their profile.

You can add a new profile or delete it if it's no longer needed. You can also make a profile the default or switch between profiles without logging out of the Keyavi app Lightweight software interface that provides a user-friendly, intuitive interaction with Keyavi's technology. It allows users to create new and open intelligent files and manage settings easily.. When you log out of the Keyavi app, you'll no longer have it available to you from the taskbar.

You won’t be able to rename a profile once it’s added. You’ll need to delete it and add it again using the organization code for that profile.

Add a Profile

Each profile you create will be associated with a unique organization code.

To create a new profile:

  1. Click the profile picture, select Profiles, and then select Manage.

  2. Click Add.

  3. Enter a descriptive and meaningful profile name, for example, Default or Keyavi USA. You can create multiple profiles, so keep the naming meaningful but straightforward.

  4. Paste the organization code you received from your Keyavi administrator.

  5. Click Save. The profile is automatically set as your default profile. When you have multiple profiles, you'll be able to specify which profile you want as the default.

If you’re unable to locate the organization code, contact your support team or Keyavi User or Security Administrator. They can provide you with the organization code. You’ll provide the organization code only once as it’s part of the setup.

Switch Profiles

If you have more than one profile, you can switch between them without logging out of Keyavi.

To switch profiles, click the profile picture, select Profiles, and then choose a profile.

Set a Default Profile

If you have more than one profile, you can select one as your default profile and switch between them without logging out of Keyavi. When you first open Keyavi, it automatically logs you into the default profile.

To set a default profile:

  1. Click the profile picture, select Profiles, and then select Manage.

  2. Select the profile you want as the default, and then click Default.

  3. Click Back to return to Keyavi.

Delete a Profile

When you delete a profile, you'll lose access to all the data associated with the profile. If you have more than one profile and delete the default profile, the next profile becomes the default. If you delete the only profile, you'll be logged out of Keyavi.

To delete a profile:

  1. Click the profile picture, select Profiles, and then select Manage.

  2. Select the profile you want to delete, and then click Delete.

  3. Click Confirm to delete the profile.
