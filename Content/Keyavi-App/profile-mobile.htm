Manage Profiles on Mobile

Profiles define your connection to an organization’s protected data. Each profile is associated with an organization code. You can share Keyavi-protected data only with other users who use the same organization code in their profile. You can only have one profile at a time on a mobile device.

Add a Profile

Before adding a new profile, you must delete the existing profile. Each profile you create will be associated with a unique organization code.

To create a new profile:

  1. Open the menu , and then tap Profile.

  2. If you have an existing profile, delete it by tapping Delete .

  3. Tap Add.

  4. Enter a descriptive and meaningful profile name, for example, Default or Keyavi USA. You can create multiple profiles, so keep the naming meaningful but straightforward.

  5. Paste the organization code you received from your Keyavi administrator.

    If you are already set up on Windows or Mac you can scan the org code. Go to the Settings page on your computer, and then click Configure Mobile. On your mobile device, tap Scan Org Code and scan the code in Keyavi.

  6. ClickAdd.

  7. Sign in using the information provided by your Keyavi administrator.

    Depending on how the organization configured Keyavi, you'll sign in with your organization's credentials or have the option to choose which credentials you use.

If you’re unable to locate the organization code, contact your support team or Keyavi User or Security Administrator. They can provide you with the organization code. You’ll provide the organization code only once as it’s part of the setup.

Delete a Profile

When you delete a profile, you'll lose access to all the data associated with the profile.

To delete a profile:

  1. Open the menu , and then tap Open.

  2. Tap Delete.
