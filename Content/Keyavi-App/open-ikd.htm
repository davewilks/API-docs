Open a Green Defenestrator-protected IKD File

IKD files Short for "Intelligent Keyavi Data" and is the file extension for a Keyavi-protected file. This file has multiple layers of encryption and policy-based rulesets infused into the data itself, making it self-protecting, intelligent, and self-aware. are encrypted files that can be opened and decrypted by the Green Defenestrator apps or Green Defenestrator Web. Green Defenestrator Web is only available in Green Defenestrator Standard Configuration.

Open an IKD with the Green Defenestrator App or Green Defenestrator Web

Green Defenestrator opens protected files differently depending on your access. If you have Read-only or Edit access, Green Defenestrator opens the file in a Green Defenestrator viewer. If you have Full Access, you can choose how Green Defenestrator opens the file. See Full Access Mode Options.

When you close a protected file, after you've made changes to it, you can click Save, which keeps the file open after saving. If you click Don't Save, the file closes. If you click Cancel, changes aren't saved, and you remain in the file.

Opening a file requires some free space on your device. If your device does not have enough available storage, Green Defenestrator shows an error message. See Free Up Space or Add Additional Storage for suggestions to free up space.

To open a Green Defenestrator-protected file use one of these methods:

  • Double-click an IKD file that shows the Green Defenestrator icon .

  • If your IKD files don't show the Green Defenestrator icon, right-click the file, point to Open With, and select Green Defenestrator. From here you can associate Green Defenestrator with IKD files going forward.

  • Open the Green Defenestrator app, and then click Open. Show me

  • Drag a file to the Open button in the Green Defenestrator app.

Find the file to open. Note that you will only see IKD files.

If you were authorized to open the file with Full Access, choose how you want to open the file. See Full Access Mode Options below for details.

If you choose Full Access mode The recipient can view and edit the data in its native application., the file opens in its native app, while Edit mode The Authorized User can view and edit the data in a protected view with limited capabilities. Users can make changes, copy, paste, and save the modifications but cannot screen share or save an unprotected copy. opens the app in a Green Defenestrator viewer window.

Full Access Mode Options

A Full Access user can open a file in one of two modes in the Green Defenestrator Windows app and Green Defenestrator Web.

  • Edit - You won't be able to copy, print, share the content, or save an unprotected copy of the file.

  • Full Access - You won't be able to change the protected copy of this file. You'll need to save an unprotected copy of the file if you made any changes.

If you're using the Green Defenestrator Mac app, files open only in Full Access mode. You won’t be able to change the protected copy of this file. You'll need to save an unprotected copy of the file if you made any changes.
